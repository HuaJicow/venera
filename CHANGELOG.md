# Changelog

## Unreleased

### Added
- **Comic sources: drag-to-reorder.** Sources now follow a custom order (with
  alphabetical fallback for any not yet ordered). A reorder button in the comic
  source page app bar opens a drag-and-drop list.
- **Explore / Categories: quick tab reordering.** Long-press a tab (mobile) or
  right-click it (desktop) to move it left, right, to the start, or to the end —
  no need to open a separate dialog.
- **Home page: option to hide comic covers.** A new "Show covers on home page"
  switch in Settings → Appearance lets you hide the cover thumbnails on the
  History and Local cards, so potentially sensitive covers are not exposed at a
  glance. Covers stay visible everywhere else; open the History/Local page to
  view them.

### Changed
- **Explore / Categories: moved the page manager to the top bar.** The "+ Add"
  button at the far right of the tab bar has been replaced with an "Edit" button
  next to the search action in the top bar. The edit dialog still supports
  adding, removing, and drag-reordering pages — but it is now always reachable
  without scrolling the tab bar.
